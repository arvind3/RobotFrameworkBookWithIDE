## Concept Explanation

The capstone combines all concepts into a nested, multi-domain project. It includes 20+ files, layered resources, Python helpers, variables, fixtures, and domain suites.

```mermaid
flowchart TB
  A[suites/main.robot] --> B[suites/auth/*.robot]
  A --> C[suites/orders/*.robot]
  A --> D[suites/reports/*.robot]
  B --> E[resources/auth.resource]
  C --> F[resources/orders.resource]
  D --> G[resources/reports.resource]
  E --> H[libraries/*.py]
  F --> H
  G --> H
```
