# Dashboard KPI Priorities

Use these top KPIs for portfolio and book-level performance.

## Core acquisition and engagement

1. Total users
2. Sessions
3. Engaged sessions
4. Engagement rate

## Audience composition

1. Users by country
2. Users by region
3. Users by device category (desktop/mobile/tablet)

## Content performance

1. Top chapters by engaged sessions
2. Chapter completion rate (`chapter_complete` / `chapter_view`)
3. Chapter trend over time

## Interaction performance

1. Code copy rate (`code_copy` / `chapter_view`)
2. TOC interaction rate (`toc_interaction` / `chapter_view`)

## Filters for all dashboards

- Date range
- `book_id`
- `content_group`
- Device category
- Country
- Region
