## Why This Page Exists

This book is designed to stay practical and opinionated, but its guidance is anchored to official Robot Framework sources and respected community documentation.

Use this page as your primary source index when you want to go deeper.

## How This Book Uses These Sources

1. Syntax, semantics, and execution model are aligned with the official User Guide.
2. Naming and layout recommendations are aligned with the Style Guide.
3. Chapter exercises intentionally reinforce variable handling, reusable keywords, and deterministic assertion patterns from official guidance.
4. Enterprise chapters map official recommendations to realistic project structures and CI flow.

## Recommendation

Read each chapter in this book first, then open the related source links at the end of that chapter to deepen your understanding with official documentation.
