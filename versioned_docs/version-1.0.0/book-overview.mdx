## What This Book Is

This book is a practical, browser-first learning path for Robot Framework.
Every chapter combines:

- concise concept teaching
- runnable multi-file examples
- an in-browser IDE + execution flow
- guided exercises and failure patterns

## Who This Book Is For

| Persona | You Will Get |
| --- | --- |
| QA Engineer / SDET | A maintainable Robot architecture with reusable keywords and scalable suites |
| Developer | Clear patterns for integrating Robot + Python libraries in real projects |
| Team Lead / Architect | Enterprise folder conventions, quality guardrails, and CI-ready practices |

## How To Use This Book

1. Read the chapter scenario and concept section.
2. Run the baseline example in the playground.
3. Make one focused change and run again.
4. Compare output, then validate against common mistakes.
5. Move to the next chapter only after the exercise passes.
